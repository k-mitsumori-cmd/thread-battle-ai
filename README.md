# 2chスレッド風・論破レスバAIゲーム

2chスレッド風のUIと文体で、ユーザーの投稿に対してAI住民が論破・レスバしてくるチャット型ゲームです。

## 🎮 特徴

- **2ch風UI**: 黒背景・緑系テキストのBBS風デザイン
- **複数住民演出**: 複数のAI住民が入れ替わりでレス
- **自動レス生成**: ユーザーの投稿に対して3〜6レス程度の論破レスを自動生成
- **2ch特有の文体**: 「ｗ」「草」「お前さぁ…」「はい論破」などの2ch特有の表現
- **安全な煽り**: 人格攻撃・差別・暴力表現は禁止。ソフトなブラックジョーク寄り

## 🚀 使い方

### ローカルで実行

1. このリポジトリをクローンまたはダウンロード
2. 依存関係をインストール：

```bash
npm install
```

3. 以下のコマンドでローカルサーバーを起動：

```bash
npx serve .
```

または、`index.html`を直接ブラウザで開くことも可能です。

4. ブラウザで `http://localhost:3000` にアクセス

**注意**: ローカル環境ではAPIが動作しないため、フォールバックレス（パターンベース）が表示されます。

### Vercelでデプロイ

1. Vercelにログイン
2. このリポジトリをインポート
3. **環境変数を設定**（重要！）:
   - Vercelダッシュボードでプロジェクトを開く
   - 「Settings」→「Environment Variables」を開く
   - 変数名: `OPENAI_API_KEY`
   - 値: あなたのOpenAI APIキー
4. 自動的にデプロイされます

**環境変数がない場合**: APIキーが設定されていない場合は、フォールバックレス（パターンベース）が表示されます。

## 🎯 機能説明

### AIによるレス生成

このアプリは**OpenAI API（GPT-4o-mini）**を使用して、ユーザーの投稿内容を分析し、2ch風の論破レスを自動生成します。

- **実際のAI思考**: ユーザーの投稿内容を理解し、論理的に論破
- **2ch特有の文体**: 「ｗ」「草」「お前さぁ…」「はい論破」などの表現を使用
- **コンテキスト理解**: 投稿内容に応じた適切なレスを生成
- **複数住民の演出**: 異なる口調・スタイルの住民がレス

### レス形式

各レスは以下の形式で表示されます：

```
1 ：風吹けば名無し：2025/01/01(月) 12:00:01.23：ID:xxxxxxx
お前さぁ…本気で言ってんの？ｗ
```

### AI住民キャラクター

- **風吹けば名無し**: 煽り系（aggressive: 0.8）
- **名無しのゴッキー**: 論理的論破系（logical: 0.8）
- **名無しさん＠お腹いっぱい**: 皮肉系（sarcastic: 0.9）
- **名無しの権兵衛**: バランス型
- **名無しのインテリ**: 論理的論破系（logical: 0.9）
- **名無しの一般人**: バランス型

### レス生成ロジック

- ユーザーの投稿に対して、3〜6レス程度のAI住民のレスを生成
- 最初のレスは直接ユーザーへのツッコミ・論破
- 中間のレスは住民同士の揉めや別視点での論破
- 最後のレスはまとめレス
- **フォールバック機能**: APIキーがない場合やエラー時は、パターンベースのレスを表示

### 特別演出

- 1000レス目に到達すると特別な演出が表示されます

## 🔧 カスタマイズ

### 住民キャラクターの追加

`app.js`の`residents`配列と`residentStyles`配列に新しいキャラクターを追加できます。

### レスパターンの追加

`app.js`の`createResponse`メソッド内のパターン配列（`aggressivePatterns`、`logicalPatterns`、`sarcasticPatterns`など）に新しいレスパターンを追加できます。
